## 📌 Overview
A complete school enrollment management system built with:
- JDBC for database connectivity
- MySQL for data persistence
- Log4j2 for structured logging
- Maven for dependency management
- Lombok for cleaner code

## 🚀 Features
- **Student Management**: Create, read, update, and delete student records
- **Course Management**: Manage course catalog and details
- **Enrollment System**: Handle student enrollments in courses
- **Data Validation**: Input validation for all operations
- **Comprehensive Logging**: Detailed activity logging

## 🛠️ Prerequisites
- Java 21 or higher
- MySQL 8.0 or higher
- Maven 3.8 or higher
- Lombok
